# EV Supply Chain Emission Simulator

This repository contains the dataset and the script files for the EV supply chain emission simulator. It also contains the optimization model cost matrices and optimization script.

The repository directory has two main folders:
1. **Simulation Model:** Contains relevant data and script files for the emission simulator.
2. **Optimization Model:** Contains the cost matrices and script files for optimization.

Additionally, the repository contains the figures script and data which were extracted from the simulation outputs.
